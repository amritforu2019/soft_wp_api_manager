 cURL Example

                    

                    

                        
You can also test using cURL or PowerShell:

                        
                        
PowerShell:

                        
$body = @{
    api_key = "YOUR_API_KEY"
    mobile_number = "1234567890"
    message_text = "Test message"
    priority = "normal"
} | ConvertTo-Json

Invoke-RestMethod -Uri "http://localhost/soft_wp_api_manager/api/send_message.php" -Method Post -Body $body -ContentType "application/json"

                        
                        
cURL:

                        
curl -X POST http://localhost/soft_wp_api_manager/api/send_message.php \
-H "Content-Type: application/json" \
-d '{"api_key":"YOUR_API_KEY","mobile_number":"1234567890","message_text":"Test message","priority":"normal"}'
